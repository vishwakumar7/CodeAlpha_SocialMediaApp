# CodeAlpha Social Media Platform

A modern social media web application developed as part of the **CodeAlpha Full Stack Development Internship**.  
This platform allows users to register, log in, create posts, like posts, comment, and interact through a clean and responsive user interface inspired by popular social media applications such as Instagram and Facebook.
